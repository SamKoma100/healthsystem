# HealthConnect SL

HealthConnect SL is a responsive Health Service website for Sierra Leone. It was created for the DCOMP204 Web Design 1 project brief and includes HTML, CSS, responsive design, JavaScript interaction, images/media, a navigation system, and a working appointment/contact form demo.

## What to Download

Download the whole `health-service-website` folder. It contains:

- `index.html` - main website page
- `styles.css` - responsive visual design
- `script.js` - JavaScript interactions
- `assets/` - local image/media assets
- `docs/project-documentation.md` - project write-up for printed documentation

## How to Open Locally

1. Open the `health-service-website` folder.
2. Double-click `index.html`.
3. The website will open in your browser.

No server is required because this is a static HTML, CSS, and JavaScript website.

## Website Features

- Responsive layout for mobile, tablet, and desktop
- Sticky navigation and mobile menu
- Local images/media
- Clinic directory with district filter
- Health tips slider
- Basic symptom guidance interaction
- Appointment request form with confirmation message
- Emergency guidance modal
- Documentation page for project submission

## Step-by-Step GitHub Pages Deployment

1. Go to [GitHub](https://github.com/) and sign in.
2. Click **New repository**.
3. Name the repository, for example `healthconnect-sl`.
4. Keep it public if your lecturer needs to access the link.
5. Click **Create repository**.
6. Upload all files from inside the `health-service-website` folder.
7. Commit the uploaded files.
8. Open the repository **Settings**.
9. Go to **Pages**.
10. Under **Build and deployment**, choose **Deploy from a branch**.
11. Select branch `main` and folder `/root`.
12. Click **Save**.
13. Wait for GitHub to generate the live link.
14. Copy the GitHub Pages URL and include it in your printed documentation.
