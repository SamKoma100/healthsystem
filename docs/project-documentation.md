# HealthConnect SL Project Documentation

## 1. Problem Description

In Sierra Leone, many people delay seeking medical care because they do not always know which clinic to visit, what services are available, or what warning signs require urgent attention. This can affect families in urban and rural communities, especially during malaria season, maternal health emergencies, child illness, and dehydration cases.

HealthConnect SL is a web-based digital solution that provides simple health service information in one place. The website helps users find example clinics by district, learn prevention tips, request appointments, and read emergency guidance.

## 2. Project Objective

The objective of this project is to build a functional, responsive health service website that supports a real societal need in Sierra Leone. The website applies HTML, CSS, responsive design, basic JavaScript, image/media use, form handling, and GitHub Pages deployment.

## 3. Target Users

- Patients looking for nearby health services
- Parents seeking child health and immunization information
- Pregnant women needing maternal care guidance
- Students and workers who need quick clinic information
- Community health volunteers sharing prevention messages

## 4. System Design

The website is designed as a static front-end web solution.

- `index.html` contains the page structure and content.
- `styles.css` contains the layout, color system, responsive design, and visual presentation.
- `script.js` contains JavaScript for the mobile menu, clinic filtering, health tips slider, symptom guidance, emergency modal, and appointment confirmation.
- `assets/` contains local website images.

## 5. Main Website Sections

1. Home hero section with project name and call-to-action buttons.
2. Problem section explaining the health service challenge.
3. Services section showing clinic finder, maternal care, and medicine support.
4. Clinic directory with district filter.
5. Health tips carousel.
6. Symptom guidance form.
7. Appointment request form.
8. Emergency support modal.
9. Footer with documentation link.

## 6. Digital Public Good Principles

HealthConnect SL supports public benefit because it focuses on health access and community information. The solution is easy to share, low-cost to host, mobile-friendly, and can be expanded with real clinic data, multilingual content, SMS integration, and Ministry of Health resources.

## 7. Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive design with media queries
- GitHub Pages for deployment
